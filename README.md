Bootstrap Components written using JADE mixins for Meteor
==================

[Bootstrap](http://getbootstrap.com/) using [Jade](http://jade-lang.com) packaged for [Meteor](https://www.meteor.com)

## Install

```bash
$ meteor add rajasegar:jade-bootstrap
```
